Vitamin C-serum – hvorfor alle bruker det

Vitamin C (10–20 %) kan gi jevnere hudtone og glød. Se etter mørk flaske, stabil formel og pH 3–4. Bruk om morgenen under solkrem; start rolig for å unngå irritasjon.
